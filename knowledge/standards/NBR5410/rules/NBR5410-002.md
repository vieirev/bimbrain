# NBR5410-002 — Identificação de painéis elétricos

## Objetivo

Garantir que todos os painéis elétricos do projeto possuam identificação válida e única, conforme os requisitos de rastreabilidade da NBR 5410.

## Descrição da verificação

A regra verifica duas condições:

1. Se existem painéis elétricos com nome vazio (elementos que deveriam ser identificados mas não possuem nenhum nome atribuído);
2. Se existem painéis com nomes duplicados (dois ou mais painéis compartilhando exatamente o mesmo nome).

Ambas as condições comprometem a correta identificação dos quadros elétricos no projeto.

## Elementos Revit utilizados

- `FamilyInstance` com categoria `OST_ElectricalEquipment` (painéis/quadros)

## Limitações

- A regra não valida nomenclaturas por convenção (ex.: "QDL-01" vs "QD-01")
- A regra não verifica a consistência entre o nome do painel e o circuito associado
- Modelos vinculados (Revit Links) não são analisados

## Critério de aprovação

Todos os painéis elétricos possuem nome preenchido e único.

## Critério de reprovação

Existe pelo menos um painel com nome vazio ou pelo menos dois painéis com o mesmo nome.

## Observações

A implementação reutiliza duas fontes existentes: `DuplicatePanelNameRule` para detectar nomes duplicados e `PanelAnalyzer` para localizar painéis com nome vazio. Nenhuma lógica de varredura do modelo é duplicada.
