# US-0003 — Consultar quantidade de tomadas

## Objetivo
Permitir que o usuário pergunte "Quantas tomadas existem?" e receba a contagem real de tomadas do modelo Revit aberto.

## Critério de Aceite
- Ao perguntar "Quantas tomadas existem?" (ou variações de capitalização/pontuação), o BIMBrain responde com o número correto de tomadas.
- Qualquer outra pergunta retorna: "Pergunta ainda não suportada pelo BIMBrain."

## Resultado Esperado
"Foram encontradas XX tomadas."
