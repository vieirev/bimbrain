# US-0016

## Título

Verificação de integridade do modelo elétrico

## Objetivo

Adicionar capacidade de verificar a integridade do modelo elétrico, identificando tomadas, luminárias, interruptores e painéis sem circuitos associados.

## Critério de Aceite

- Todas as consultas existentes continuam funcionando.
- Integridade do modelo pode ser consultada.
- Elementos sem circuito são identificados.
- Painéis sem circuitos são identificados.
- Código reutiliza os analyzers existentes.
- Build sem erros.
