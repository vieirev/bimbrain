# US-0026

## Título

RuleResult com identificação dos elementos afetados

## Objetivo

Adicionar à classe RuleResult a capacidade de registrar quais ElementIds originaram cada inconsistência, preparando a Rules Engine para localização visual de elementos com problemas.

## Critério de Aceite

- Build sem erros.
- Nenhuma funcionalidade existente muda.
- Todas as regras continuam funcionando.
- Cada RuleResult identifica exatamente quais elementos originaram o problema.

## Resultado Esperado

| Regra | ElementId retornado |
|-------|-------------------|
| CircuitWithoutPanelRule | Id do circuit sem painel |
| PanelWithoutCircuitRule | Id do painel sem circuitos |
| ElementsWithoutCircuitRule | Ids dos elementos sem circuito |
| UnloadedLinksRule | lista vazia |
| DuplicatePanelNameRule | Ids dos painéis com nome duplicado |
| NBR5410-001 | Ids das tomadas sem circuito |
| NBR5410-002 | Ids dos painéis com identificação inválida |
