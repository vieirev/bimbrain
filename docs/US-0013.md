# US-0013

## Título

Distribuição de elementos elétricos por nível

## Objetivo

Adicionar capacidade de analisar a distribuição dos elementos elétricos por nível, permitindo consultas como "tomadas por nível", "luminárias por nível" etc. A Engine deve detectar automaticamente a categoria e retornar uma resposta estruturada.

## Critério de Aceite

- Todas as consultas existentes continuam funcionando.
- A distribuição é calculada pela Engine.
- Não utilizar IA.
- Código reutilizável em ElementLevelAnalyzer.
- Build sem erros.
