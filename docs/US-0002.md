# US-0002

## Título

Interface de consulta simples

## Objetivo

Permitir que o usuário digite uma pergunta em linguagem natural e veja a resposta na janela do BIMBrain. Ainda sem IA ou interpretação real.

## Critério de Aceite

- Visualizar o nome do projeto na janela
- Digitar uma pergunta no campo de texto
- Clicar em "Consultar"
- Receber "Pergunta recebida." se houver texto
- Receber "Digite uma pergunta." se o campo estiver vazio

## Resultado Esperado

```
BIMBrain

Projeto:
<Nome do Projeto>
──────────────────────────────
Faça uma pergunta:
[___________________________] [Consultar]

Resposta:
Pergunta recebida.
```
