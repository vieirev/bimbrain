# US-0018

## Título

Primeira regra de consistência — Circuitos sem painel

## Objetivo

Implementar a primeira regra de engenharia real utilizando a infraestrutura criada na TASK-0040. A regra identifica circuitos elétricos que não possuem painel associado (BaseEquipment nulo) e reporta cada ocorrência como um Warning.

## Critério de Aceite

- Build sem erros.
- RuleRunner executa a regra.
- Circuitos sem painel são identificados.
- Nenhuma funcionalidade existente é alterada.

## Resultado Esperado

```
src/BIMBrain/Rules/Consistency/
├── CircuitWithoutPanelRule.cs

Engine → EngineeringRule → Execute(Document) → List<RuleResult>
```

Para cada circuito sem painel:
- RuleName = "Circuitos sem painel"
- Success = false
- Severity = Warning
- Message = "Circuito '{nome}' (Id {id}) não possui painel associado."

Caso todos os circuitos estejam associados:
- RuleName = "Circuitos sem painel"
- Success = true
- Severity = Information
- Message = "Todos os circuitos possuem painel associado."
