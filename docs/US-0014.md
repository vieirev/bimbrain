# US-0014

## Título

Análise estrutural dos circuitos elétricos

## Objetivo

Adicionar capacidade de listar e analisar a estrutura dos circuitos elétricos do projeto, exibindo nome do circuito, painel associado e contagem de tomadas, luminárias e interruptores conectados.

## Critério de Aceite

- Todas as consultas existentes continuam funcionando.
- Nova consulta lista todos os circuitos.
- Exibe painel quando disponível.
- Exibe quantidade de elementos conectados.
- Exibe resumo final.
- Não utiliza IA.
- Build sem erros.
