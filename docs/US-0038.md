# US-0038 — Orquestração do Copilot

## Título

TASK-0062: Infraestrutura oficial de orquestração do BIMBrain (Foundation Engine,
EPIC-0001).

## Objetivo

Criar a infraestrutura oficial de orquestração do BIMBrain. O `CopilotOrchestrator`
apenas decide qual Engine deve atender uma solicitação, analisando o
`CopilotContext`. Nenhuma funcionalidade é executada: não chama IA, não executa
Rules, não executa consultas. Nenhuma funcionalidade, Rule, IA, consulta ou
comportamento existente é alterado.

## Critérios de Aceite

- [x] Nenhuma funcionalidade existente alterada.
- [x] Nenhuma Rule alterada.
- [x] Nenhuma IA alterada.
- [x] Nenhuma consulta alterada.
- [x] Nenhum comportamento existente alterado.
- [x] Classe apenas decide.
- [x] Nenhuma execução.

## Resultado Esperado

- Arquivos criados: `src/BIMBrain/CopilotRequestType.cs`,
  `src/BIMBrain/CopilotRoute.cs`, `src/BIMBrain/CopilotOrchestrator.cs`.
- `docs/ARCHITECTURE.md` atualizado com CopilotOrchestrator no fluxo e nos
  componentes Foundation.
- `docs/CAPABILITIES.md` atualizado com a seção Copilot Orchestration.
- O orquestrador apenas roteia; não executa nenhum Engine.
