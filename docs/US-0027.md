# US-0027 — Seleção automática dos elementos de uma regra

## Objetivo

Permitir que o BIMBrain selecione automaticamente no Revit os elementos
relacionados a um `RuleResult`, facilitando a inspeção visual das
inconsistências encontradas.

## Regras de Negócio

- Se `AffectedElements` estiver vazio, `Select` retorna `false` e não altera
  a seleção atual.
- Se `AffectedElements` contiver elementos, `Select` substitui a seleção
  atual pelos `ElementId`s da lista.
- `ClearSelection` limpa a seleção independentemente do estado atual.
- Nenhuma transação é criada ou necessária.
- Nenhum elemento do modelo é alterado.

## API

```csharp
namespace BIMBrain
{
    public class RuleSelectionService
    {
        // Seleciona os elementos afetados de um RuleResult.
        // Retorna true se a seleção foi aplicada, false se a lista estava vazia.
        public bool Select(UIDocument uiDoc, RuleResult result);

        // Limpa a seleção atual do documento.
        public void ClearSelection(UIDocument uiDoc);
    }
}
```

## Exemplo de uso

```csharp
var service = new RuleSelectionService();
var results = RuleRunner.RunAll(document, rules);

foreach (var result in results)
{
    if (!result.Success)
        service.Select(uiDoc, result);
}
```

## Critérios de Aceite

- Nenhuma regra existente é alterada.
- Nenhuma consulta existente é alterada.
- Nenhuma interface é alterada.
- Nenhum elemento do modelo é modificado.
- Build sem erros.
- O serviço pode ser reutilizado futuramente pela interface.
