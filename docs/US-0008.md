# US-0008

## Título

Melhoria da interface do BIMBrain (UX)

## Objetivo

Transformar a janela do BIMBrain em uma interface agradável para uso diário dentro do Revit, com layout organizado em quatro regiões, redimensionamento livre, resposta em TextBox rolável, histórico funcional com limpar e reexibir, e barra de status inferior.

## Fluxo Principal

1. Usuário abre o BIMBrain no Revit.
2. Janela 900x700 é exibida com quatro regiões.
3. Região 1: nome do projeto e modelo ativo.
4. Região 2: campo de pergunta (Enter para consultar) + botões Consultar e Testar IA.
5. Região 3: resposta (TextBox rolável à esquerda) e histórico (à direita, 280px).
6. Região 4: barra inferior com status, tempo e modelos carregados.
7. Usuário digita pergunta, pressiona Enter ou clique em Consultar.
8. Resposta aparece no TextBox; histórico registra a entrada.
9. Usuário pode clicar em item do histórico para reexibir resposta.
10. Usuário pode limpar o histórico ou copiar para área de transferência.
11. Janela pode ser redimensionada livremente; layout se reorganiza.

## Critério de Aceite

- Todas as consultas existentes continuam funcionando.
- A interface pode ser redimensionada (mínimo 700x500).
- Respostas grandes ficam totalmente legíveis (TextBox com scroll).
- Histórico continua funcionando com cópia e limpeza.
- Selecionar um item do histórico reapresenta sua resposta.
- Tempo da consulta continua sendo exibido na resposta e na barra de status.
- ModelContext não mostra documentos duplicados.
- Campo de pergunta aceita Enter para consultar.
- Barra inferior mostra status, tempo e quantidade de modelos.
- Nenhuma funcionalidade existente é removida.
