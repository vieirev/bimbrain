# US-0022

## Título

Quinta regra de consistência — Painéis com nomes duplicados

## Objetivo

Implementar regra que identifica painéis elétricos que compartilham exatamente o mesmo nome, garantindo que cada painel possua identificação única no projeto.

## Critério de Aceite

- Build sem erros.
- Um RuleResult por nome duplicado.
- Um RuleResult de sucesso quando não houver duplicidade.
- Nenhuma funcionalidade existente é alterada.

## Resultado Esperado

Para cada nome duplicado:
- RuleName = "Nomes duplicados de painéis"
- Success = false
- Severity = Warning
- Message = "Foram encontrados {N} painéis com o nome '{nome}'."

Caso todos os painéis tenham nomes únicos:
- RuleName = "Nomes duplicados de painéis"
- Success = true
- Severity = Information
- Message = "Todos os painéis possuem identificação única."
