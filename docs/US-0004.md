# US-0004

## Título

Apresentação visual organizada das respostas

## Objetivo

Melhorar a exibição das respostas na janela do BIMBrain, substituindo o texto simples por um layout estruturado com seções visuais separadas (status, projeto, pergunta, resultado, tempo de consulta).

## Critério de Aceite

- Todas as 14 consultas existentes continuam funcionando.
- A interface exibe a resposta em formato visualmente organizado.
- O tempo da consulta é exibido em milissegundos.
- Nenhuma funcionalidade existente é removida.

## Resultado Esperado

```
[Consulta concluída]
‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾
Projeto: <Nome do Projeto>

Pergunta: Quantas tomadas existem?

Resultado:
  Foram encontradas 42 tomadas.

Tempo: 28 ms
```

Em caso de erro:

```
[Consulta não realizada]
‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾
Motivo:
Pergunta ainda não suportada pelo BIMBrain.
```

Estado inicial:

```
[Aguardando consulta...]
‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾
Nenhuma pergunta informada.
Digite uma pergunta antes de consultar.
```
