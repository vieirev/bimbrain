# US-0031 — Reorganização documental em plataforma de engenharia assistida por IA

## Título

TASK-0055: Reorganização documental — BIMBrain como plataforma de engenharia assistida por IA.

## Objetivo

Reorganizar toda a documentação do projeto para refletir a evolução do BIMBrain
de um MVP para uma plataforma de engenharia assistida por IA. Nenhum código,
funcionalidade, classe, regra ou comportamento deve ser alterado. A TASK é
exclusivamente documental.

## Critérios de Aceite

- [x] Nenhum arquivo `.cs` modificado.
- [x] Nenhuma funcionalidade alterada.
- [x] Nenhuma regra modificada.
- [x] Arquitetura documentada (ARCHITECTURE.md).
- [x] Visão do produto documentada (VISION.md).
- [x] Motores documentados (ENGINES.md).
- [x] Disciplinas documentadas (DISCIPLINES.md).
- [x] Roadmap reorganizado (ROADMAP.md).
- [x] Documentação consistente entre todos os arquivos.

## Resultado Esperado

- Arquivos criados: VISION.md, ARCHITECTURE.md, PRODUCT_PRINCIPLES.md,
  ENGINES.md, DISCIPLINES.md, FUTURE.md, US-0031.md.
- Arquivos modificados: ROADMAP.md, README.md, AGENTS.md.
- O projeto passa a referenciar explicitamente os Engines, as disciplinas e os
  princípios de produto, mantendo consistência com CAPABILITIES.md e a base de
  conhecimento existente.
