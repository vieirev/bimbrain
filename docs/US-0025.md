# US-0025

## Título

Catálogo de RuleSets

## Objetivo

Introduzir os conceitos de RuleSet e RuleCatalog para organizar as regras de engenharia em grupos lógicos. Nenhuma regra nova é criada, nenhuma regra existente é alterada.

## Critério de Aceite

- Build sem erros.
- Nenhuma regra existente muda.
- Nenhum comportamento muda.
- Apenas arquitetura.

## Resultado Esperado

- `src/BIMBrain/Rules/RuleSet.cs` — classe com Name, Description, Rules
- `src/BIMBrain/Rules/RuleCatalog.cs` — catálogo estático com RuleSet NBR 5410 (NBR5410-001 e NBR5410-002)
- `knowledge/standards/NBR5410/README.md` — estrutura da norma adicionada
