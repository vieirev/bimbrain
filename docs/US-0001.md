# US-0001

## Título

Exibir informações básicas do modelo Revit

## Objetivo

Ao clicar no botão "Abrir BIMBrain", o plugin deve ler o modelo ativo e exibir nome do projeto, total de elementos, quantidade de vistas e níveis.

## Critério de Aceite

A janela do BIMBrain exibe as informações reais do modelo aberto no Revit.

## Resultado Esperado

```
BIMBrain

Projeto:
<Nome do Projeto>

Elementos: <N>
Vistas: <N>    Níveis: <N>
```
