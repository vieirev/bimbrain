# US-0017

## Título

Infraestrutura para execução de regras de engenharia

## Objetivo

Criar a infraestrutura base para execução de regras de engenharia no BIMBrain. Esta etapa define as classes abstratas e o executor que permitirão a implementação futura de normas técnicas (NBR, IEC etc.) sem impacto na estrutura existente.

## Critério de Aceite

- Nenhuma funcionalidade existente muda.
- Nenhuma consulta nova é criada.
- Nenhuma norma técnica é implementada.
- RuleRunner executa regras genéricas.
- Build sem erros.

## Resultado Esperado

```
src/BIMBrain/Rules/
├── EngineeringRule.cs  (classe abstrata)
├── RuleResult.cs       (resultado de execução)
├── RuleSeverity.cs     (enum: Information, Warning, Error)
└── RuleRunner.cs       (executor de regras)
```
