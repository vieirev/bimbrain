# US-0049 — Classificação por categoria Revit

## Título

TASK-0073: Primeira camada de classificação do Classification Engine, baseada
exclusivamente na categoria Revit (Classification Engine, EPIC-0005).

## Objetivo

Implementar `ElementClassifier.Classify(Element)` mapeando
`element.Category.BuiltInCategory` para `ElementClassificationType`. Sem uso de
Family, Symbol, Tipo, Parâmetros ou IA.

## Critérios de Aceite

- [x] Painéis (`OST_ElectricalEquipment`) → Panel.
- [x] Circuitos (`OST_ElectricalCircuit`) → Circuit.
- [x] Conduítes (`OST_Conduit`) → Conduit.
- [x] Bandejas (`OST_CableTray`) → CableTray.
- [x] Rooms (`OST_Rooms`) → Room.
- [x] Levels (`OST_Levels`) → Level.
- [x] Demais categorias → Unknown.
- [x] Nenhum uso de nomes de famílias.
- [x] Nenhuma IA.
- [x] Nenhuma Rule alterada.
