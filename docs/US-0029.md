# US-0029 — Infraestrutura para destaque visual de elementos

## Objetivo

Criar a API pública para destacar visualmente no Revit os elementos
relacionados a um `RuleResult`, permitindo que inconsistências sejam
facilmente identificadas na vista ativa.

## Regras de Negócio

- Nesta primeira versão, os métodos lançam `NotImplementedException`.
- A implementação real com `OverrideGraphicSettings` será feita em TASK futura.
- Nenhum elemento do modelo é alterado.
- Nenhuma informação é gravada no modelo.

## API

```csharp
namespace BIMBrain
{
    public class RuleHighlightService
    {
        // Aplica destaque visual nos elementos afetados de um RuleResult.
        bool Highlight(UIDocument uiDoc, View view, RuleResult result);

        // Remove o destaque visual dos elementos na vista.
        bool Clear(UIDocument uiDoc, View view);
    }
}
```

## Critérios de Aceite

- Build sem erros.
- Nenhuma funcionalidade existente alterada.
- Serviço criado e pronto para futura implementação.
