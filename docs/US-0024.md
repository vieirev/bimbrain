# US-0024

## Título

Segunda regra normativa — NBR5410-002: Identificação de painéis elétricos

## Objetivo

Implementar a segunda regra documentada da base normativa, verificando se todos os painéis elétricos possuem nome preenchido e único. A regra reutiliza `DuplicatePanelNameRule` para nomes duplicados e `PanelAnalyzer` para nomes vazios.

## Critério de Aceite

- Build sem erros.
- Nenhuma lógica duplicada.
- Reutilização das regras existentes.
- Knowledge atualizado.
- INDEX atualizado.

## Resultado Esperado

- `knowledge/standards/NBR5410/rules/NBR5410-002.md` — documentação da regra
- `src/BIMBrain/Rules/Standards/NBR5410/NBR5410_002_IdentifiedPanelsRule.cs` — implementação
- Exemplo de sucesso: "Todos os painéis possuem identificação válida e única."
- Exemplo de reprovação: "Foram encontrados painéis com identificação inválida."
