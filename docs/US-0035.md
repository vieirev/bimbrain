# US-0035 — Explicação de elemento via Project Graph

## Título

TASK-0059: Primeiro caso de uso completo do Project Graph — explicação de
elemento (Model Engine, EPIC-0002).

## Objetivo

Criar o primeiro caso de uso completo combinando `ProjectGraph`,
`ProjectGraphQuery` e `ProjectImpactAnalyzer` para entregar uma explicação
completa de um elemento do modelo. Nenhuma Rule, IA ou consulta existente é
alterada. Toda a operação ocorre exclusivamente sobre o grafo.

## Critérios de Aceite

- [x] Nenhuma consulta alterada.
- [x] Nenhuma Rule alterada.
- [x] Nenhuma IA alterada.
- [x] Nenhum acesso ao Revit.
- [x] Utiliza exclusivamente o Graph (via ProjectGraphQuery e ProjectImpactAnalyzer).

## Resultado Esperado

- Arquivos criados: `src/BIMBrain/Graph/ElementExplanation.cs`,
  `src/BIMBrain/Graph/ElementExplanationService.cs`.
- `docs/ARCHITECTURE.md` atualizado com ElementExplanationService após
  ProjectImpactAnalyzer.
- `docs/CAPABILITIES.md` atualizado com a seção Element Explanation.
- O serviço recebe `ProjectGraph`, `ProjectGraphQuery` e `ProjectImpactAnalyzer`,
  e nunca acessa `Document`, `FilteredElementCollector` ou Revit API.
