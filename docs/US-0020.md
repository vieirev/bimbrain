# US-0020

## Título

Terceira regra de consistência — Elementos elétricos sem circuito

## Objetivo

Implementar a terceira regra de engenharia utilizando a infraestrutura da Rules Engine. A regra identifica tomadas, interruptores e luminárias que não pertencem a nenhum circuito elétrico. O ModelIntegrityAnalyzer existente é refatorado para reutilizar esta regra, eliminando lógica duplicada.

## Critério de Aceite

- Build sem erros.
- Nenhuma funcionalidade existente muda.
- ModelIntegrityAnalyzer reutiliza a Rules Engine.
- Não existe lógica duplicada.
- Um RuleResult por categoria inconsistente.
- Um RuleResult de sucesso quando tudo estiver correto.

## Resultado Esperado

```
src/BIMBrain/Rules/Consistency/
├── ElementsWithoutCircuitRule.cs
```

Para cada categoria com elementos sem circuito:
- RuleName = "Elementos sem circuito"
- Success = false
- Severity = Warning
- Message = "Foram encontradas {N} {categoria} sem circuito."

Caso todos os elementos possuam circuito:
- RuleName = "Elementos sem circuito"
- Success = true
- Severity = Information
- Message = "Todos os elementos analisados possuem circuito."

ModelIntegrityAnalyzer.Analyze() passa a executar a regra via RuleRunner e extrair os counts das mensagens, mantendo o contrato público inalterado.
