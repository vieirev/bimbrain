# US-0030 — Camada de orquestração das ações sobre RuleResult

## Objetivo

Criar uma camada única que encapsule os serviços de seleção, navegação e
destaque visual, oferecendo uma interface simplificada para a interface do
BIMBrain.

## Regras de Negócio

- Cada método delega para o serviço correspondente sem lógica adicional.
- Nenhuma alteração é realizada no modelo Revit.
- Nenhuma transação é criada.

## API

```csharp
namespace BIMBrain
{
    public class RuleActionService
    {
        bool Select(UIDocument uiDoc, RuleResult result);
        bool Navigate(UIDocument uiDoc, RuleResult result);
        bool Highlight(UIDocument uiDoc, View view, RuleResult result);
        bool ClearHighlight(UIDocument uiDoc, View view);
    }
}
```

## Exemplo de uso

```csharp
var actions = new RuleActionService();
var results = RuleRunner.RunAll(document, rules);

foreach (var result in results)
{
    if (!result.Success)
    {
        actions.Select(uiDoc, result);
        actions.Navigate(uiDoc, result);
    }
}
```

## Critérios de Aceite

- Build sem erros.
- Nenhum comportamento alterado.
- Apenas encapsulamento dos serviços existentes.
- Nenhuma duplicação de código.
