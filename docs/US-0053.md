# US-0053 — Electrical Analyzer baseado em classificação

## Título

TASK-0077: Remover de `ElectricalCircuitAnalyzer` a dependência de categorias
específicas do Revit para identificar tomadas, interruptores e luminárias
(Model Engine + Classification Engine, EPIC-0005).

## Objetivo

Fazer toda identificação de tomada/interruptor/luminária passar obrigatoriamente
pelo `ElementClassifier`. `BuiltInCategory` pode localizar elementos, mas nunca
mais decide o que o elemento representa.

## Critérios de Aceite

- [x] Consultas existentes retornam os mesmos resultados.
- [x] Nenhuma Rule alterada.
- [x] Nenhuma IA.
- [x] Nenhuma janela alterada.
- [x] Nenhum comportamento visível muda.
- [x] Identificação de tomada/interruptor/luminária só via Classification Engine.

## Detalhes

- `ElectricalCircuitAnalyzer.Analyze()` agora usa `_classifier.Classify(element)`
  e faz `switch(classification.Classification)`:
  `Outlet` → tomadas++, `Switch` → interruptores++, `LightingFixture` →
  luminárias++.
- `BuiltInCategory` continua usado apenas para coletar `ElectricalSystem` (localização).
- `GetConnectedElementIds()` inalterado (coleção de ids, sem classificação).
- Nenhuma lógica de negócio nem resposta muda.
