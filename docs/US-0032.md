# US-0032 — Infraestrutura inicial do Project Graph

## Título

TASK-0056: Infraestrutura inicial do Project Graph (Model Engine, EPIC-0002).

## Objetivo

Criar a infraestrutura inicial do Project Graph — a representação navegável do
modelo Revit. Esta TASK cria apenas a infraestrutura. Nenhuma consulta, regra ou
comportamento existente é alterado; a IA ainda não utiliza o grafo.

## Critérios de Aceite

- [x] Nenhuma consulta existente alterada.
- [x] Nenhuma regra modificada.
- [x] Nenhum comportamento existente alterado.
- [x] Graph criado (GraphNode, GraphEdge, GraphRelation, ProjectGraph).
- [x] Builder criado (ProjectGraphBuilder).
- [x] Nós básicos funcionando (Projeto, Modelo, Nível, Painel, Circuito).
- [x] Relacionamentos básicos funcionando (Contains, FedBy).
- [x] Arquitetura atualizada (docs/ARCHITECTURE.md, docs/CAPABILITIES.md).

## Resultado Esperado

- Arquivos criados em `src/BIMBrain/Graph/`: GraphNode.cs, GraphEdge.cs,
  ProjectGraph.cs, ProjectGraphBuilder.cs (GraphRelation definido em GraphEdge.cs).
- `docs/ARCHITECTURE.md` atualizado com o Graph entre Model Engine e Rule Engine.
- `docs/CAPABILITIES.md` atualizado com a seção Project Graph.
- O Builder reutiliza ModelContext, PanelAnalyzer e ElectricalCircuitAnalyzer, sem
  duplicar coletores existentes, e não lança exceções.
