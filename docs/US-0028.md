# US-0028 — Navegação automática até elementos de uma regra

## Objetivo

Permitir que o BIMBrain navegue automaticamente no Revit até os elementos
relacionados a um `RuleResult`, facilitando a inspeção visual das
inconsistências encontradas.

## Regras de Negócio

- Se `AffectedElements` estiver vazio, `ZoomTo` e `ZoomToFirst` retornam
  `false` e não alteram a vista atual.
- Se `AffectedElements` contiver elementos, `ZoomTo` navega para todos os
  elementos da lista.
- `ZoomToFirst` navega apenas para o primeiro elemento da lista.
- Nenhuma transação é criada ou necessária.
- Nenhum elemento do modelo é alterado.

## API

```csharp
namespace BIMBrain
{
    public class RuleNavigationService
    {
        // Navega para todos os elementos afetados de um RuleResult.
        // Retorna true se a navegação foi realizada, false se a lista estava vazia.
        public bool ZoomTo(UIDocument uiDoc, RuleResult result);

        // Navega apenas para o primeiro elemento afetado.
        // Retorna true se a navegação foi realizada, false se a lista estava vazia.
        public bool ZoomToFirst(UIDocument uiDoc, RuleResult result);
    }
}
```

## Exemplo de uso

```csharp
var nav = new RuleNavigationService();
var results = RuleRunner.RunAll(document, rules);

foreach (var result in results)
{
    if (!result.Success)
        nav.ZoomTo(uiDoc, result);  // navega para todos os elementos da inconsistência
}
```

## Critérios de Aceite

- Build sem erros.
- Nenhuma regra modificada.
- Nenhuma consulta modificada.
- Nenhuma alteração no modelo.
- Serviço reutilizável por futuras interfaces.
