# US-0043 — Execução do RuleSet NBR 5410 pela Ribbon

## Título

TASK-0067: Primeira execução real de uma norma diretamente pela Ribbon (Rule
Engine, EPIC-0003).

## Objetivo

Implementar o comando `NBR 5410` (grupo Engenharia da Ribbon) que executa todas
as regras cadastradas no RuleSet `NBR 5410` e apresenta um relatório organizado.

## Critérios de Aceite

- [x] Botão `NBR 5410` funcional.
- [x] `RuleCatalog` utilizado para localizar `"NBR 5410"`.
- [x] `RuleRunner` utilizado para executar todas as regras do RuleSet.
- [x] Todas as regras do RuleSet executadas (NBR5410-001, NBR5410-002).
- [x] `TaskDialog` apresenta resumo organizado.
- [x] Nenhuma regra alterada.
- [x] Nenhuma IA utilizada (sem Ollama, sem `QuestionProcessor`).

## Fluxo

```
Ribbon (NBR 5410)
  ↓ RunNBR5410Command
  ↓ RuleCatalog.All.FirstOrDefault(rs => rs.Name == "NBR 5410")
  ↓ RuleRunner(ruleSet.Rules).RunAll(doc)
  ↓ TaskDialog
```

## Detalhes Técnicos

- `RunNBR5410Command` (`IExternalCommand`) substitui `PlaceholderCommand` no
  botão `NBR 5410`.
- Usa obrigatoriamente `RuleCatalog` para localizar o RuleSet; as regras não são
  instanciadas manualmente.
- `RuleRunner.RunAll(doc)` achata os resultados de todas as regras
  (`SelectMany`).
- `BuildReport` agrega: Regras executadas, Regras aprovadas, Warnings, Errors,
  status por regra (`✔`/`⚠`/`❌`) e Resultado Geral.
- Status por regra: `✔` quando todos os resultados da regra são `Success`;
  `❌` se houver algum `Error`; `⚠` caso contrário (warnings).
- Resultado Geral: `MODELO APROVADO` se não houver warnings nem errors;
  `MODELO COM INCONSISTÊNCIAS` caso contrário.
