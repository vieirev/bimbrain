# US-0037 — Resolução de contexto do Copilot

## Título

TASK-0061: Camada oficial de resolução de contexto do BIMBrain (Foundation Engine,
EPIC-0001).

## Objetivo

Criar a camada oficial de resolução de contexto do BIMBrain. O `CopilotContext`
centraliza pergunta, seleção, projeto e grafo em um único objeto, antes da
execução de qualquer funcionalidade. Não responde perguntas, não usa IA, não
executa Rules e não faz consultas — apenas agrega contexto. Nenhuma funcionalidade,
Rule, IA, consulta ou navegação existente é alterada.

## Critérios de Aceite

- [x] Nenhuma funcionalidade existente alterada.
- [x] Nenhuma Rule alterada.
- [x] Nenhuma IA alterada.
- [x] Nenhuma consulta alterada.
- [x] Nenhuma navegação duplicada.
- [x] Apenas agregação de contexto.

## Resultado Esperado

- Arquivos criados: `src/BIMBrain/CopilotContext.cs`,
  `src/BIMBrain/CopilotContextBuilder.cs`.
- `docs/ARCHITECTURE.md` atualizado com CopilotContextBuilder/CopilotContext no
  fluxo e nos componentes Foundation.
- `docs/CAPABILITIES.md` atualizado com a seção Copilot Context.
- O builder reutiliza `SelectionContextService` e `ProjectGraph`; não duplica
  navegação e não executa regras nem IA.
