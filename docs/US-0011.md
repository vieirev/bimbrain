# US-0011

## Título

Reorganização do planejamento com EPICs

## Objetivo

Reorganizar o planejamento do BIMBrain utilizando EPICs como estrutura organizacional de alto nível, preparando a evolução do projeto para que todas as futuras TASKs pertençam obrigatoriamente a um EPIC.

## Critério de Aceite

- Estrutura `docs/epics/` criada com 6 EPICs documentados.
- ROADMAP reorganizado como roadmap de alto nível apenas com EPICs.
- TASK.md atualizado com coluna EPIC relacionando todas as tasks ao EPIC-0001.
- MILESTONES relacionado aos EPICs.
- AGENTS.md atualizado com regra sobre EPICs.
- Nenhum arquivo em `src/` modificado.
- Nenhuma funcionalidade alterada.

## Resultado Esperado

```
docs/epics/
├── EPIC-0001.md  Foundation       (✅ Concluído)
├── EPIC-0002.md  BIM Engine       (🔄 Em andamento)
├── EPIC-0003.md  Rules Engine     (📋 Planejado)
├── EPIC-0004.md  AI Engine        (📋 Planejado)
├── EPIC-0005.md  Engineering Know. (📋 Planejado)
└── EPIC-0006.md  Production       (📋 Planejado)
```

ROADMAP passa a conter apenas a tabela de EPICs com status.
TASK.md ganha coluna EPIC.
MILESTONES ganha coluna EPIC.
AGENTS.md ganha regra de dependência com EPIC.
