# US-0019

## Título

Segunda regra de consistência — Painéis sem circuitos

## Objetivo

Implementar a segunda regra de engenharia utilizando a infraestrutura da Rules Engine. A regra utiliza o PanelAnalyzer existente para identificar painéis elétricos que não possuem nenhum circuito associado.

## Critério de Aceite

- Build sem erros.
- Regra reutiliza PanelAnalyzer.
- Um RuleResult por painel inconsistente.
- Um RuleResult de sucesso quando não houver inconsistências.
- Nenhuma funcionalidade existente é alterada.

## Resultado Esperado

Para cada painel sem circuitos:
- RuleName = "Painéis sem circuitos"
- Success = false
- Severity = Warning
- Message = "Painel '{nome}' não possui nenhum circuito associado."

Caso todos os painéis possuam circuitos:
- RuleName = "Painéis sem circuitos"
- Success = true
- Severity = Information
- Message = "Todos os painéis possuem ao menos um circuito associado."
