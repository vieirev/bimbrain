# US-0021

## Título

Quarta regra de consistência — Modelos vinculados descarregados

## Objetivo

Implementar regra que identifica modelos Revit Link descarregados no projeto, reutilizando o ModelContext para descoberta de vínculos sem duplicar lógica.

## Critério de Aceite

- Build sem erros.
- Regra reutiliza ModelContext.
- Um RuleResult por modelo descarregado.
- Um RuleResult de sucesso quando todos estiverem carregados.
- Nenhuma funcionalidade existente é alterada.

## Resultado Esperado

Para cada modelo descarregado:
- RuleName = "Modelos descarregados"
- Success = false
- Severity = Warning
- Message = "O modelo '{nome}' está descarregado."

Caso todos os modelos estejam carregados:
- RuleName = "Modelos descarregados"
- Success = true
- Severity = Information
- Message = "Todos os modelos vinculados estão carregados."
