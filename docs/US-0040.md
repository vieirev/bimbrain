# US-0040 — Integração do pipeline do Copilot ao fluxo

## Título

TASK-0064: Integração oficial do pipeline do Copilot ao fluxo do BIMBrain
(Foundation Engine, EPIC-0001).

## Objetivo

Integrar o novo pipeline do Copilot ao fluxo atual do BIMBrain. A execução passa
obrigatoriamente por `CopilotExecutor` → `CopilotContextBuilder` →
`CopilotOrchestrator` → `QuestionProcessor`. A interface não muda e o
comportamento observado pelo usuário permanece exatamente igual. Alteração
exclusivamente arquitetural.

## Critérios de Aceite

- [x] Todas as consultas atuais continuam funcionando.
- [x] Nenhuma resposta muda.
- [x] Nenhuma Rule muda.
- [x] Nenhuma IA muda.
- [x] Nenhuma janela muda.
- [x] CopilotExecutor torna-se o ponto único de entrada.
- [x] QuestionProcessor deixa de ser chamado diretamente pela UI.

## Resultado Esperado

- `QuestionProcessor` ganha o overload `ProcessQuestionAsync(CopilotContext)`,
  que extrai a pergunta e delega ao método `string` existente (nenhum branch
  removido).
- `MainWindow` constrói o `CopilotExecutor` (graph + query + impact + explanation
  + selection + context builder + orchestrator) e passa a invocar
  `CopilotExecutor.Execute(question)`, entregando `execution.Context` a
  `QuestionProcessor`.
- `Command` passa o `UIDocument` ao `MainWindow` para viabilizar o pipeline.
- `docs/ARCHITECTURE.md` e `docs/CAPABILITIES.md` atualizados; `docs/US-0040.md`
  criado.
