# US-0007

## Título

Contexto do projeto — descoberta de modelos vinculados

## Objetivo

Fazer o BIMBrain compreender o contexto completo do projeto Revit, identificando todos os modelos vinculados (Revit Links) carregados, seus respectivos documentos e a quantidade de ambientes (Rooms) em cada um.

## Fluxo Principal

1. Usuário pergunta "modelos carregados", "informações dos modelos" ou "contexto do projeto".
2. BIMBrain cria um `ModelContext` a partir do documento ativo.
3. `ModelContext` varre o documento ativo em busca de `RevitLinkInstance`.
4. Para cada instância encontrada, obtém o `LinkDocument` correspondente.
5. Para cada link carregado, conta os ambientes (Rooms) via `FilteredElementCollector`.
6. BIMBrain exibe o resumo com: projeto ativo, modelos encontrados, status e quantidade de Rooms.

## Critério de Aceite

- Build sem erros.
- O BIMBrain identifica corretamente o documento ativo.
- O BIMBrain identifica todos os Revit Links carregados.
- O BIMBrain informa a quantidade de Rooms existente em cada documento.
- Links descarregados são identificados como "Descarregado" sem tentativa de acesso a elementos.
- Nenhuma funcionalidade existente deixa de funcionar.
