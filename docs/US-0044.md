# US-0044 — Janela de resultados de regras

## Título

TASK-0068: Substituir o `TaskDialog` da execução do RuleSet por uma janela
profissional de resultados (Rule Engine, EPIC-0003).

## Objetivo

Criar `RuleResultsWindow`, a janela oficial de visualização de resultados de
regras do BIMBrain, e integrá-la à execução do RuleSet NBR 5410 no lugar do
`TaskDialog`.

## Critérios de Aceite

- [x] Executar `NBR 5410`.
- [x] Janela abre.
- [x] Lista todas as regras (cada `RuleResult`).
- [x] Selecionar uma regra mostra detalhes (Nome, Severidade, Mensagem, Qtd. elementos).
- [x] Botão `Selecionar` funciona (`RuleActionService.Select`).
- [x] Botão `Navegar` funciona (`RuleActionService.Navigate`).
- [x] Nenhuma Rule alterada.
- [x] Nenhuma IA utilizada.

## Detalhes Técnicos

- `RuleResultsWindow` (WPF code-only, namespace `BIMBrain.UI`) recebe
  `UIDocument` e `List<RuleResult>`.
- Lista superior: `ListBox` com uma linha por `RuleResult` (Ícone, Nome,
  Status, Mensagem resumida via `TextTrimming`).
- Painel inferior: `Border` com os detalhes da regra selecionada; botões
  `Selecionar`/`Navegar` ficam desabilitados quando `AffectedElements` está vazio.
- `Selecionar` → `RuleActionService.Select(uiDoc, result)`; `Navegar` →
  `RuleActionService.Navigate(uiDoc, result)`. Aviso caso não haja elementos.
- `Exportar` → placeholder (`TaskDialog` "Funcionalidade em desenvolvimento.").
- `RunNBR5410Command` agora constrói a janela e chama `ShowDialog()` em vez do
  `TaskDialog`; mantém `RuleCatalog` + `RuleRunner`.
- Usa `RuleResult.AffectedElements` diretamente; não recalcula nem reconsulta o modelo.

## Arquivos

- `src/BIMBrain/UI/RuleResultsWindow.cs` — nova janela (criada).
- `src/BIMBrain/RunNBR5410Command.cs` — passa a abrir `RuleResultsWindow`

- `docs/CAPABILITIES.md` — seção `Rule Results Window` adicionada.
- `docs/US-0044.md` — criado.
