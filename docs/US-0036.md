# US-0036 — Infraestrutura de contexto da seleção

## Título

TASK-0060: Infraestrutura oficial de contexto da seleção do usuário (Foundation
Engine, EPIC-0001).

## Objetivo

Criar a infraestrutura oficial de contexto da seleção do usuário. A seleção apenas
obtém o `ElementId` inicial; toda descoberta de relações ocorre pelo `ProjectGraph`
via `ProjectGraphQuery`, `ProjectImpactAnalyzer` e `ElementExplanationService`. Esta
camada será consumida futuramente por IA, Rules, Automation, Coordination e
Document Engine. Nenhuma consulta, Rule, IA, janela ou comportamento existente é
alterado.

## Critérios de Aceite

- [x] Nenhuma Rule alterada.
- [x] Nenhuma consulta alterada.
- [x] Nenhuma IA alterada.
- [x] Nenhuma janela alterada.
- [x] Reutilização máxima dos serviços existentes.
- [x] Nenhuma lógica duplicada.

## Resultado Esperado

- Arquivos criados: `src/BIMBrain/SelectionContext.cs`,
  `src/BIMBrain/SelectionContextService.cs`.
- `docs/ARCHITECTURE.md` atualizado com SelectionContextService (fluxo e
  componentes Foundation).
- `docs/CAPABILITIES.md` atualizado com a seção Selection Context.
- O serviço recebe `UIDocument`, `ProjectGraph`, `ProjectGraphQuery`,
  `ProjectImpactAnalyzer` e `ElementExplanationService`, e reutiliza todos sem
  duplicar navegação.
