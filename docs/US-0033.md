# US-0033 — Mecanismo de navegação do Project Graph

## Título

TASK-0057: Mecanismo oficial de navegação do Project Graph (Model Engine,
EPIC-0002).

## Objetivo

Criar a camada de navegação do Project Graph. O grafo já representa o modelo;
agora ele precisa ser consultável. Nenhuma consulta, Rule ou comportamento
existente é alterado. A navegação ocorre inteiramente sobre o grafo, sem acesso
direto ao Revit. A classe é genérica e reutilizável por qualquer disciplina.

- [x] Nenhuma Rule modificada.
- [x] Nenhuma IA modificada.
- [x] Nenhum comportamento alterado.
- [x] ProjectGraph continua independente do Revit.
- [x] Query totalmente baseada no Graph.
- [x] BFS funcionando.
