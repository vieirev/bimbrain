# US-0012

## Título

Diagnóstico avançado dos modelos vinculados

## Objetivo

Evoluir a análise dos modelos vinculados (Revit Links), transformando a consulta "contexto do projeto" em um diagnóstico técnico completo que exibe resumo quantitativo, detalhamento por modelo e diagnóstico de integridade (links descarregados, duplicados, etc.).

## Critério de Aceite

- Todas as consultas existentes continuam funcionando.
- Consulta "contexto do projeto" exibe o resumo dos modelos.
- Quantidade de elementos exibida por documento.
- Quantidade de níveis exibida por documento.
- Diagnóstico identifica links descarregados.
- Diagnóstico identifica links duplicados.
- Nenhuma consulta existente é removida.
- Build sem erros.
